Engineered for Power Users

More than a launcher. Astera provides system tuning, cinematic curation, and metadata scraping pipelines built on top of high-performance native registry scanners.

Universal Registry Scanner

Scans and compiles local games registry directories across Steam, Epic Games, GOG Galaxy, EA, Ubisoft, Xbox, and custom standalone directories, aggregating your entire library in seconds.

Real-time Telemetry & Diagnostics

Monitor hardware metrics like active CPU cores utilization, GPU temperature, and active system memory usage directly in your hub dashboard without overlay lags.

Hardware Spec Tuning Hook

Automatically toggles Windows power schemes to High Performance on game start, returning to Balanced on shutdown. Flush standby RAM garbage collection before boot cycles.

Cinematic 3D Tilt Spotlight

An interactive spotlight that bends to mouse coordinates, dynamic Unsplash color-tinted particle fields, and an automated background description web scraper pipeline.

Advanced Pipeline Configuration

Define custom launch arguments (e.g. -novid -fps_max 144), inject custom environment variables, or execute pre-launch shell scripts prior to loading titles.

Stunning Profiles & Backdrops

Make the launcher your own. Create user profile blocks, select futuristic vector badges, adjust dynamic RGB accent hex codes, and toggle high-fidelity shader backdrops.

Developer & Install Setup

Follow the steps below to setup, compile, or contribute code to the Astera project repository.

01

CLONE AND INSTALL

First install packages and local electron requirements. Make sure Node.js is installed.

npm install
02

RUN IN LOCAL DEV MODE

Launch the hot-reloaded Electron window to test development builds instantly.

npm run dev
03

COMPILE EXECUTABLES

Run installer scripts to compile and package production installer packages.

npm run build:win

Packages output into the dist/ folder and copy setup executables to the root directory on completion.
